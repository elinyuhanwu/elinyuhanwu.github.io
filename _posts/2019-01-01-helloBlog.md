**Jekyll**

本地調適在Terminal輸入
```
gem install jekyll
```

如果是使用Mac系統有設定密碼時需要經過授權需輸入
```
sudo gem install jekyll
```

使用時需要另外安裝`jekyll-paginate plug-in`並寫入`_config.yml`中：
```
gem install jekyll-paginate
```

在本地伺服器預覽：
```
jekyll serve
``` 
或是
```
jekyll serve --watch
```
